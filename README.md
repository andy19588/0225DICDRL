# 梁仲龍 — Personal Website

A sleek, Squarespace-inspired single-page personal website built with pure HTML, CSS & JavaScript.

> *"A website makes it real."*

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Dark Theme** | Premium dark UI with subtle animated gradient orbs and grain texture overlay |
| 🧭 **Sticky Navbar** | Frosted-glass navigation bar that blurs on scroll (glassmorphism) |
| 🔤 **Marquee Banner** | Continuously scrolling text — *DESIGN ✦ CREATE ✦ INSPIRE ✦ BUILD ✦ DREAM* |
| 📊 **Stats Cards** | Animated stat blocks — ∞ 可能性 · 100% 熱情 · 24/7 靈感 |
| ⏰ **Live Clock** | Real-time digital clock with date display, plus a smooth analog clock |
| 🎞️ **Scroll Animations** | Fade-in elements triggered by scroll via `IntersectionObserver` |
| 📱 **Fully Responsive** | Adapts seamlessly to desktop, tablet, and mobile screens |
| 🔡 **Premium Typography** | Google Fonts — [Inter](https://fonts.google.com/specimen/Inter) + [Noto Sans TC](https://fonts.google.com/noto/specimen/Noto+Sans+TC) |

## 🚀 Getting Started

No build tools or server required — just open the HTML file:

```bash
# Clone the repository
git clone https://github.com/andy19588/0225DICDRL.git

# Open in browser
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

Or visit the live site via **GitHub Pages** (if enabled):  
🔗 `https://andy19588.github.io/0225DICDRL/`
